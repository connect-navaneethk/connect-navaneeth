03. Creations

Featured Projects

01

Human Follower Car

An Arduino-based robotic vehicle designed to follow a person automatically while maintaining a suitable distance, demonstrating practical use of sensors, motor control, and autonomous movement.

Working Principle & Features

  • Uses Ultrasonic & IR sensor array to map human presence in front.
  • Embedded obstacle avoidance algorithms prevent crashes.
  • Continuous servo tracking allows real-time adjustment of motor rotation.
  • Designed with low-power DC gear motors and custom chassis.
Arduino Uno Ultrasonic Sensor DC Gear Motors Servo Motor Motor Driver Module
Demo / GitHub
02

Smart Parking Ground

A parking management system developed to monitor parking slot occupancy and provide users with real-time availability via a 16x2 I2C LCD display and sound announcements using a DF Player mini speaker setup.

Working Principle & Features

  • Utilizes infrared occupancy sensors in each slot for instant detection.
  • Interfaced using I2C protocol to simplify LCD wire routing.
  • DF Player Mini triggers audio clips to guide drivers audibly.
  • Centralized Arduino microcontroller manages logical flow and states.
Arduino Uno Occupancy Sensors 16x2 LCD DF Player Mini Embedded Logic
Demo / GitHub
03

DIY PCB LED Blinker

A self-fabricated LED blinking circuit configured in astable mode using a 555 Timer IC. Manufactured on a custom PCB using the etching method to master soldering, circuit assembly, and practical hardware prototyping.

Working Principle & Features

  • 555 Timer IC operating in astable mode generates continuous pulse train.
  • Features variable blinking speeds via potentiometer adjustments.
  • Hand-crafted custom board etching using ferric chloride solutions.
  • Integrated audio buzzer feedback synchronized with blinking pulses.
555 Timer IC Custom PCB Etching Potentiometer Buzzer Soldering
